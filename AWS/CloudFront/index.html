AWS CloudFront

Host a file through CloudFront

February 6, 2021 by Maximilian Lever

If you are reading this Tutorial, it is assumed you have access to files hosted in an AWS S3 bucket.
  1. Open CloudFront from AWS Management console.
  2. Click
  4. In “Origin Domain Name” select the S3 bucket the files are in.
  5. “Origin Path” is where you specify the folder where the files are in the Bucket.
    E.G. /assets
  6. Select “Yes” on “Enable Origin Shield”.
  7. Select the region where you need your files to be available.
  8. Select “No” on “Restrict Bucket Access” so the files can only be accessed through the CloudFront.
  9. Leave the rest as default and click
  10. Wait until status is “Deployed”.
  11. Once deployed you will have a domain name.
    E.G. uniqueCode.cloudfront.net* uniqueCode will be replaced with a code specific to your distribution.
  12. The files are accessed in a web browser with the domain name for your distribution.
    E.G. http://d23hbb1mr42qgd.cloudfront.net/assets/test.html
