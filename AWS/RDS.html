AWS RDS Set up

September 9, 2020 by Maximilian Lever

  1. Select Standard Create
  2. Under Engine options select MYSQL
  3. Select MySQL Community
  4. Leave Version as default
  5. Template Production
  6. Name your DB instance
  7. Fill Credentials and make a note of them
  8. Select instance size to suit your needs
  9. Select Allocated storage
  10. Deselect Enable storage autoscaling
  11. Select Create a standby instance
  12. Under Additional connectivity configuration in Connectivity tab select yes for Public access
  13. Select Password authentication
  14. Create database
  15. Edit inbound rules
  16. Add
    HTTP TCP 80 Source = Anywhere //This rule is for your instance to be publicly accessed by browser.
  17. Add
    SSH TCP 22 Source = Anywhere //This rule is to allow GitBash to SSH and SCP to your instance.
  18. Add
    All Traffic TCP 22 Source = Anywhere //This rule is to allows all traffic to your instance.
