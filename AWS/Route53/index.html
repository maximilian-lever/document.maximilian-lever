Registering a domain

September 12, 2020 by Maximilian Lever

  1. Open Route 53 console.
  2. Under Register domain search the domain you would like and clickcheck
  3. It will show you if the domain name is available or not:
    • If not available you can change the endinge.g..com,.co.uk,.eu
    • If available add to cart and click next
  4. Fill in contact information and click continue
  5. If you would like to enable auto renew make sure “enable” ischecked under “Do you want to automatically renew your domain?”
  6. Check “I have read and agree to the AWS Domain Name RegistrationAgreement”
  7. Click Complete order

Registering a subdomain

September 04, 2021 by Maximilian Lever

  1. Open the Route 53 console.
  2. In the navigation pane, choose Hosted zones.
  3. Choose Create hosted zone.
  4. In the right pane, enter the name of the subdomain (such as some.example.com).
  5. For Type, accept the default value of Public hosted zone.
  6. Choose Create hosted zone.
