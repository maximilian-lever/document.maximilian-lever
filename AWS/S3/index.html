Hosting from S3

February 6, 2021 by Maximilian Lever

  1. Open s3 from AWS Management console
  2. Click
  3. Fill in:
    • Bucket name (Unique name)
    • Region (Closest region to where your files will be accesses)
  4. Leave all configure options as default
  5. Make sure “Block all public access” is unchecked
  6. Under Management system permissions select Grant Amazon S3 Log Delivery group write access to this bucket
  7. Click
  8. Select newly created bucket
  9. Under permissions tab click
  10. In the policy editor insert and replace BUCKET NAME with the unique name chosen in step 4: 
                        {
    
                                  "Version": "2012-10-17",
    
                                  "Statement": [
    
                                      {
    
                                          "Sid": "PublicReadGetObject",
    
                                          "Effect": "Allow",
    
                                          "Principal": "*",
    
                                          "Action": "s3:GetObject",
    
                                          "Resource": "arn:aws:s3:::BUCKET NAME/*"
    
                                      }
    
                                  ]
    
                              }
  11. Under Properties tab click Static web hosting
  12. Select Use this bucket to host a website
  13. The URL will be at the top of the Static web hosting box next to endpoint:
    E.G. Endpoint : http://example.s3-website.eu-west-2.amazonaws.com
