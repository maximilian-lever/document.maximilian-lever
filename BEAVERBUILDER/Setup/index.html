Beaver Builder Setup

Setup

September 9, 2020 by Maximilian Lever

Step 1:

Go to PAGES tab

trash existing pages

add new Homepage

Step 2:

Go to PLUGINS tab

Install:

bb-builder-pro.zip

bb-theme-builder.zip

Then enter the licence key for Beaver Builder Pro

Step 3:

Go to APPEARANCE tab and then THEMES

Install:

bb-theme.zip

bb-theme-child.zip

Step 4:

Go to appearance customize. Follow the paths below to edit the settings:

Settings>Homepage Settings>Homepage = static page

Header>Header Layout>Layout = none

Footer>Footer Widgets Layout>Footer Widgets Display = none

Footer>Footer Layout>Layout = none

Step 5:

Click on ‘Pages tab’

Step 6:

Add New Page

Step 7:

Add title ‘Homepage’

Publish

Step 8:

Launch Beaver Builder
