Import & Export Template

September 9, 2020 by Maximilian Lever

Export:

Step 1:

Go to visit site and hem open beaver builder

Step 2:

Ctrl + J to save template and name the template and click ‘Save’

Export from Ultimate Beaver builder:

https://www.ultimatebeaver.com/free-premium-beaver-builder-templates/?

Step 1:

Go to ‘SETTINGS TAB’

Step 2:

Click on ‘Template Cloud’

Step 3:

Find the template you want to work on and select ‘Install’

Import: Step 1:

Go to WordPress dashboard and on the ‘TOOLS’ tab select Import

Step 2:

At the bottom install ‘WordPress Importer’ and once installed run the importer

Step 3:

Choose the file that was downloaded when you saved the template and then click ‘Upload file and Import’

Step 4:

Go back to site and open Beaver Builder and click on the + in the top right corner next to ‘Done’

Step 5:

Click the ‘Templates tab’ and then select ‘Saved templates’ in the drop down

Step 6:

Select the template that you exported and then select ‘replace existing layout’
