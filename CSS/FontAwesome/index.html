CSS Font Awesome

Font Awesome

March 6th, 2021 by Maximilian Lever

  1. Go to fontawesome.com and pick an icon.
  2. Insert the Font Awesome CDN in to the <head>
    <head>
    <link rel="stylesheet" id="font-awesome-5-css" href="http://www.maximilian-lever.com/vendor/fontawesome-free/css/all.min.css" type="text/css" media="all">
    </head>
  3. Click Start Using This Icon
  4. Copy the icon code in to your webpage.
    e.g.
    <i class="fab fa-linkedin"></i>
