Bootstrap Grid system

March 5th, 2021 by Maximilian Lever

The Bootstrap grid system is made up of rows and columns

  1. In the <body> create a div with the bootstrap class ‘row’.
  2. In the ‘row’ create as many div tags you need with the bootstrap class ‘col’.
    <body>
    <div class="row">
    <div class="col-sm">< /div>
    <div class="col-sm">< /div>
    < /div>
    < /body>

Grid options

Bootstrap’s grid system can adapt across all six default breakpoints, and any breakpoints you customize. The six default grid tiers are as follow:

Extra small (xs) - None (auto)

Small (sm) - 540px

Medium (md) - 720px

Large (lg) - 960px

Extra large (xl) - 1140px

Extra extra large (xxl) - 1320px

Table Responsive

March 5th, 2021 by Maximilian Lever

To make a table responsive you must put a table with in a div tag using the Bootstrap class ‘table-responsive’

<div class="table-responsive">
<table class="table">
...
< /table>
< /div>
