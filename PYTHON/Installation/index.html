Python Installation

Installation

September 21, 2021 by Maximilian Lever

1.Download Python (www.python.org)

2.Run .exe

3.Tick "Add Python 3.x to PATH"

4.Select "Custom installation"

5.Ensure "pip" and "py launcher" is ticked and click next

6.Tick "Install for all users", "Associate files with Python", "Add Python to environment variables", "precompile standard library" and then Install
