Set Up

September 21, 2021 by Maximilian Lever

1.Call the webdriver from the Selenium libary using the import below.

from selenium import webdriver

2.We will now set a variable to the location of chromedriver.exe

PATH = "C:\Program Files (x86)\chromedriver.exe"# Location of Chrome webdriver

3.Create a variable which will store the webdriver.

driver = webdriver.Chrome(PATH)

4.To give the driver a website use the .get() function. This will open us a chrome window and display the Website given. (AKA a head)

driver.get('http://document.maximilian-lever.com/')

Example:

# --- SCRAPER --------------------------------------------------
from selenium import webdriver
# --------------------------------------------------------------
PATH = "C:\Program Files (x86)\chromedriver.exe"# Location of Chrome webdriver
driver = webdriver.Chrome(PATH, chrome_options=options)
driver.get('http://document.maximilian-lever/')
driver.quit()

Headless webdriver

September 21, 2021 by Maximilian Lever

1.This line we will put under the webdiver import.

from selenium.webdriver.chrome.options import Options

2.Now set the headless option

options = Options()
options.headless = True
driver = webdriver.Chrome(PATH, chrome_options=options)

Example:

# --- SCRAPER --------------------------------------------------
from selenium import webdriver
fromselenium.webdriver.chrome.options import Options
# --------------------------------------------------------------
PATH = "C:\Program Files (x86)\chromedriver.exe"# Location of Chrome webdriver
options = Options()
options.headless = True
driver = webdriver.Chrome(PATH, chrome_options=options)
driver.get('http://document.maximilian-lever.com/')
driver.quit()

Selectors

September 21, 2021 by Maximilian Lever

Find single elements.
e.g. web element with an id if there are mutiple elements for the selector you have used then the driver will stop after the first element it comes to.

find_element_by_id('ID')
find_element_by_name('NAME')
find_element_by_xpath('XPATH')
find_element_by_link_text('LINKTEXT')
find_element_by_partial_link_text('LINKTEXT')
find_element_by_tag_name('TAGNAME')
find_element_by_class_name('CLASSNAME')
find_element_by_css_selector('#SELECTOR')

Find mutiple elements.
e.g. web elements with a class

find_elements_by_name('NAME')
find_elements_by_xpath('XPATH')
find_elements_by_link_text('LINKTEXT')
find_elements_by_partial_link_text('LINKTEXT')
find_elements_by_tag_name('TAGNAME')
find_elements_by_class_name('CLASSNAME')
find_elements_by_css_selector('#SELECTOR')

When working with mutiple elements the best way to cycle through what the driver has found is below.

names = driver.find_elements_by_css_selector(.name)# Creates an array of names
for name in names:# Loops for length of array
print(name.text)# prints inner text of each web element of names array
